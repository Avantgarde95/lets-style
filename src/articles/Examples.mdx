## Examples

Basic example

```typescript
const Button1 = styled("button")`
  border: 1px solid black;
  background-color: transparent;
`;
```

<BasicExampleDemo />

Pseudo class

```typescript
const Button2 = styled("button")`
  border: 1px solid black;
  background-color: transparent;

  &:hover,
  &:active {
    background-color: skyblue;
  }
`;
```

<PseudoClassDemo />

Use other component

```typescript
const Button3 = styled(Button2)`
  font-weight: bold;
`;
```

<OtherComponentDemo />

Mix the styles

```typescript
const roundedBorders = css`
  border-radius: 10px;
`;

const putTransition = css`
  transition: background-color 1s;
`;

const Button4 = styled(Button3)`
  ${roundedBorders}
  ${putTransition}
`;
```

<MixStylesDemo />

Keyframes

```typescript
const fadeIn = keyframes`
  0% {
    opacity: 0;
  }
  50% {
    opacity: 1;
  }
  100% {
    opacity: 0;
  }
`;

const Button5 = styled(Button4)`
  animation: 2s ${fadeIn} infinite;
`;
```

<KeyframesDemo />

Use component props

```typescript
interface Button6Props {
  count: number;
  onClick: () => void;
}

const Button6 = styled(Button3)<Button6Props>`
  color: ${({ count }) => (count > 5 ? "red" : "black")};
`;
```
